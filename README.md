# рейтинг жүйесі
Бұл жоба студенттердің рейтинг жүйесін құруға арналған. Бағдарлама студенттерді GPA (орташа балл) бойынша сұрыптап, нәтижесін көрсетеді. Мақсаты — сұрыптау алгоритмдерін практикада қолдану, Comparable интерфейсін меңгеру. Әр студенттің аты мен GPA мәні енгізіліп, Collections.sort() арқылы рейтинг ретімен шығарылады.
